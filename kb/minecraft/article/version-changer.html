How to Set the Minecraft Version on Your Server

Follow these steps to set the version of your Minecraft server using our Pterodactyl Game Panel:

Step 1: Log in to the Pterodactyl Game Panel

Open your web browser and navigate to the Pterodactyl Game Panel login page. Enter your credentials and click the login button.

Step 2: Access Your Server

Once logged in, you will see a list of your servers. Click on the Minecraft server you want to configure.

Step 3: Navigate to the Server Settings

In the server dashboard, click on the "Settings" tab located in the navigation menu.

Step 4: Change the Minecraft Version

In the settings, you will find an option to select the Minecraft version. Choose the desired version from the dropdown menu.

Step 5: Save Changes and Restart the Server

After selecting the desired version, click the "Save" button. Once saved, restart your server for the changes to take effect.
